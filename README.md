# MIApps

MIApps é um software que executa aplicativos desenvolvidos em PHP para desktop Linux (Debian e Ubuntu) e Windows.

## Requerimento
- Linux Debian 12 ou superior
- Linux Ubuntu 22.04 ou superior
- Windows 10 ou superior
- Arquitetura: x64
